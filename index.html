1. PLUG SENSE: AI-Powered USB Fingerprinting & NILM 🛡️

Security layer defending endpoints against BadUSB hardware attacks via side-channel analysis.

Traditional cybersecurity cannot detect spoofed digital IDs (VID/PID) on connected devices. **PLUG SENSE** solves this by shifting validation to the physical layer. By sampling the transient DC current drawn through a customized isolated sensor, it maps out hardware footprints and classifies devices using an embedded AI inference pipeline.

Hardware: ESP32, ACS712-05B Current Sensor, 12-bit Analog-to-Digital Converter.

Software Architecture: Dockerized Backend Engine, Streamlit UI Dashboard, Python REST API.

2. SolarX: AI-Supervised Dual-Axis Solar Tracker ☀️

A smart solar panel tracker that optimizes mechanical wear versus expected optical energy yield.

Rather than blindly rotating toward any transient glare (which degrades servos), **SolarX** integrates an intelligent AI supervisory dashboard over an Arduino Nano motor matrix. It dynamically runs boundary constraints, cable-stress checks, and shadow optimization, calculating exactly whether moving is worth the physical expenditure.

Hardware Array: Arduino Nano, 4 Corner LDRs, 2 Mechanical Hobby Servo Motors.

Decisions: MOVE, SLOW AUTO, HOLD POSITION, and DIFFUSE HOLD based on light stability.

3. Professional USB-UART & ST-Link V2.1 Multi-Tool 🔌

A high-fidelity R&D test board combining full hardware flow control serial bridges with micro-architecture flashing engines.

This modular PCB interface was crafted for rapid hardware debugging. It incorporates an industry-standard serial link alongside a full-featured STM32 programmer. A custom PNP-driven re-enumeration network eliminates the physical need to unplug and replug target controllers during flash loops, streamlining iterative firmware deployments.

Silicon: FT232RL (UART Bridge), STM32F103C8T6 (ARM Cortex-M3).

Power Architecture: USB Type-C input with CC1/CC2 pull-down detection, reverse polarity protection, and a dual 3.3V/5V variable VCCIO level jumper.

LOGIC LEVEL
3.3V / 5V
INTERFACE

PCB Layer Architecture Details

Full-Flow Control USB-UART Layer (FT232RL)

Brings out standard TX/RX plus CTS, RTS, DTR, DSR, DCD, and RI lines to header pins, turning this module into a comprehensive tool for validating hardware-level communications.

Re-Enumeration (RENUM) Switching Subsystem

Leverages a PNP transistor configuration controlled by STM32's PA15 pin to programmatically pulse the 1.5kΩ pull-up on USB D+ line, letting the operating system reset connection profiles without manual disconnect.

Advanced SWD Debug & printf Stream (SWO)

Routes the Serial Wire Output (SWO) alongside default clock/reset SWD pins to the target output header, unlocking rapid telemetry reporting and non-invasive trace inspection.
